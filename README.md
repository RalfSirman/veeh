# veeh
Veeh Noten
Komma separiert: G1,G#2,A2,A#2,H4,C4,C#4,D2,D#2,E4,F2,F#4,g2,g#2,a4-c4
groß: unten, klein 1 Oktave höher 
Zahlenangabe für die Länge, z.B. C1(ganz),C2(für halbe),C4 viertel (wobei man die 4 auch weglassen kann)
C8 achtel
Akkord (2 Töne gleichzeitig): E-G
Synkope: z.B. e-
Widerholung :G,G,A,G,C,H2:
Position des Cursors im Editor wird in der Notendarstellung angezeigt
Fehler im Editor werden markiert
